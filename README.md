Not A CMS
===

Content managment system written in PHP (originally known as FrostByte)

Installation
------------

1. Clone this repo
2. Copy `settings.php.sample` to `settings.php`
3. Edit settings.php
4. (Optional) configure your web server subdomain
5. Load the page
